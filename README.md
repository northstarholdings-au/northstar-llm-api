# Contributing to NorthStar LLM API

Thank you for your interest in NorthStar LLM API.

This project provides private AI inference for Australian businesses where data sovereignty is a compliance requirement. Contributions that improve accessibility, documentation, and integration examples are welcome.

---

## How to Contribute

### Report Issues

If you encounter problems with the API — unexpected responses, authentication issues, latency problems — open a GitHub Issue with:

- Your request structure (remove your API key)
- The response you received
- The response you expected
- Your plan tier

### Suggest Improvements

Open a GitHub Issue tagged `enhancement` with:

- The problem you are trying to solve
- Your proposed solution
- The industry context if relevant (healthcare, legal, finance, government)

### Submit Integration Examples

We particularly welcome real-world integration examples in:

- Python
- JavaScript / Node.js
- PHP
- Ruby
- Go
- Any framework commonly used in Australian regulated industries

Submit as a pull request adding a file to an `/examples` directory.

## Data Sovereignty Commitment

All contributions must be consistent with our core principle: **Australian data never leaves Australian infrastructure.**

No changes that route data offshore will be accepted regardless of performance benefit.
